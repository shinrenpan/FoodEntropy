# App Icon 設計原始檔

「食熵」概念：左側整齊的綠色點陣（秩序）向右發散、變橙轉紅（混亂），
呼應「熵」——食材放著不管會隨時間劣化，並對上 App 的狀態色（🟢期限內 / 🟡快到 / 🔴過期）。
